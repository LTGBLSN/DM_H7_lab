# DM_H7_lab
达妙开发板驱动库

### 使用事项：

1. 增加DBUS、SBUS两种遥控器类型协议解析，通过编译预处理改变遥控器类型
2. 更改RM电机驱动API，将ID和CAN_channel整合进驱动函数
3. 更改达妙电机驱动API，使能函数由原先的传入整个结构体转换为传入单个结构体成员；发送函数由原先传入整个结构体转换为仅传入必要成员，可直接赋值驱动便于debug
4. 更改达妙电机接收部分，新增达妙专属协议解析函数，传入结构体指针即可完成数据更新
5. 小米电机驱动整合至达妙电机驱动内部，使用达妙相同方式、电机初始化时将电机类型设置为小米即可
6. 小米电机驱动未实测（2025.1.31）
